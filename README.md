## Getting Started

First, run the development server:

```bash
npm install
```

If you have problems installing, use this command:

```bash
npm install --legacy-peer-deps
```

Next step:

```bash
npm run dev
```

Open [http://localhost:3000/me](http://localhost:3000/me) with your browser to see the result.
